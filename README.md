# Car Simulator

This is a In-Development Project that is About Recreate IRL Vehicle Physics and Controls in a Game

## What is the Goal of this Project

The Ultimate Goal of this Project is to be the One of the most Realistic Car Simulators and offer super realistic Physics and Controls inspired by Real Cars.

## What is used in this Project

This Project is Being Built completely in Godot Engine because of its Simplicity, Node Based System and heavy Customization.
